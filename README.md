# Fizzflyer - Travel and Contant Management Website

1. Install Mysql, PHP and Phpmyadmin.
2. Make a database called Fizzfler.
3. Upload fizzfler.sql in your phpmyadmin.
4. Open (localhost/fizz-flyer) in your browser.
